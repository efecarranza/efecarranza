Profile

I am a firm believer that, as Joan Manuel Serrat would say, there is no road set for me, that I make my own road. I have passed the quarter century mark, and I can honestly say, it's been a blast. What follows is my story... so far.

I also hold a minor in Mathematics, and I like to program as you can tell by this website. I made all this from scratch using HTML, CSS and Python for the blog.
